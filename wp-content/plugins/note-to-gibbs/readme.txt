== Description ==

Display a random Note to Gibbs from author Pierre O'Rourke, as seen on [NoteToGibbs.com](http://notetogibbs.com/ "NoteToGibbs.com").

== Installation ==

1. Download the zip file and extract the files
1. Upload all the files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Once activated, a Note to Gibbs widget will be come available in your widget admin section

== Frequently Asked Questions ==

= What is Note to Gibbs? =
* "Note to Gibbs" is a series of books consisting of notes to a puppy about his first time impressions while sharing thoughts on life, challenges, love, & friendship. The books will be sold on Amazon and book stores, and accompany Gibbs and Bijou when they visit children in hospitals and hospices.
